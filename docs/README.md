# Mantenimiento VLA

Este directorio resume las rutinas del sistema.

## Mensual

1. Revisar Salud.
2. Descargar y verificar un respaldo.
3. Simular el cierre mensual.
4. Ejecutar el cierre una sola vez.
5. Revisar el resultado del portón.

## Periódico

Cada seis meses, simular el cierre de auditoría antes de archivar y limpiar pagos antiguos.

## Criterios vigentes

La información financiera continúa visible para toda la comunidad. No se añade acceso individual ni segundo factor. WhatsApp se mantiene fuera de esta entrega.
